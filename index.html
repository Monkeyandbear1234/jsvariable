VARIABLES:

A JavaScript variable is simply a name of storage location.

There aare two types of variables in Javascript : local variable and global variable.

There are some rules while declaring a JavaScript varibale ( also know as identifiers)

1. Name must start with a letter ( a to z or A to Z)

2. underscore (_), or dollar ($) sign.

3. After first letter we can use digits (0 to 9), for example value 1.

4. JavaScript variables are case sensitive for example x and X are diffrent variables.
